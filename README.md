## 🚀 Sobre o Projeto
Este é o meu portfólio pessoal, desenvolvido para apresentar meus projetos, habilidades e trajetória como **Analista de Sistemas focado em Desenvolvimento Fullstack**. O objetivo principal foi criar uma interface futurista, limpa e de alta performance, aplicando conceitos avançados de UI/UX.

> "Transformando café em código de alta performance e interfaces memoráveis."

---

## 🛠️ Tecnologias e Ferramentas
O projeto foi construído "do zero", priorizando a leveza e o controle total do código:

* **HTML5 & CSS3**: Estrutura e estilização avançada (Flexbox/Grid).
* **JavaScript (Vanilla)**: Lógica de interatividade e manipulação de DOM.
* **AOS (Animate On Scroll)**: Biblioteca utilizada para as animações de entrada durante a rolagem.
* **Figma**: Planejamento visual, criação de layouts e prototipagem.
* **Boas Práticas de UI/UX**: Foco em usabilidade, hierarquia visual e acessibilidade.
